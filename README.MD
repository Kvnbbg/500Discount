## Getting Started

To run the **Tokyo TCG Hub & Star Bazaar** interface locally:

1. **Clone** this repository:

2. Navigate to the project directory:
    ```bash
    cd 500Discount
    ```
3. Open the `index.html` file in your browser to launch the application.
