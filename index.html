Deploy to Vercel in Minutes

  1. Create an index.html with scripts, add style.css, and keep assets in the repo root.
  2. Optional: add vercel.json to rewrite all routes to index.html.
  3. Push to GitHub and import the repo in Vercel for instant static deploys.

Quick Customization Tips

  • Swap in free GLTF cars (Sketchfab) and adjust materials for city lighting.
  • Overlay campaign stats and delivery milestones directly on the HUD.
  • Test locally with npx serve, then push to redeploy.
